# 02_tech_stack.md (技術の地図)

## 言語・フレームワーク
- **Frontend**: Next.js (App Router), React, Tailwind CSS
- **Backend**: Next.js Server Actions

## インフラ・プラットフォーム
- **Hosting**: Vercel
- **Database / Auth**: Supabase (PostgreSQL)
- **Payment**: Stripe

- **Lucide React** (アイコンが必要な場合)
- **(追加予定)**: 認知機能モデル化用の数学ライブラリ等
